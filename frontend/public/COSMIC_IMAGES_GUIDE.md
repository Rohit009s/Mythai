# Cosmic Background Images Setup

## Required Images

Place 5 divine cosmic power images in the `frontend/public/` directory with these names:

1. `cosmic1.jpg` - First cosmic image
2. `cosmic2.jpg` - Second cosmic image  
3. `cosmic3.jpg` - Third cosmic image
4. `cosmic4.jpg` - Fourth cosmic image
5. `cosmic5.jpg` - Fifth cosmic image

## Image Specifications

- **Format:** JPG or PNG
- **Resolution:** 1920x1080 or higher (Full HD+)
- **Theme:** Divine cosmic power, nebulas, galaxies, spiritual energy
- **Colors:** Rich purples, blues, golds, cosmic colors
- **Style:** Ethereal, mystical, divine

## Recommended Image Sources

1. **NASA Images:** https://images.nasa.gov/
   - Search for: nebula, galaxy, cosmic, supernova
   
2. **Unsplash:** https://unsplash.com/
   - Search for: cosmic, nebula, space, universe, divine
   
3. **Pexels:** https://www.pexels.com/
   - Search for: cosmic, space, galaxy, stars

## Animation Behavior

The images will:
- Fade smoothly between each other (3 second transition)
- Change every 8 seconds
- Slowly zoom in/out (20 second cycle)
- Create a meditative, divine atmosphere
- Have subtle twinkling star overlay

## Current Setup

The CosmicBackground component is already integrated into your app and will automatically cycle through these images with smooth, slow transitions perfect for a divine, meditative experience.
